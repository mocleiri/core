Dojo Bubble Feature

DojoBubble is a simple widget displaying a bubble on a given html id or on an other component.


 

Java Sources


package wicket.contrib.dojo.examples;

import wicket.PageParameters;
import wicket.ajax.AjaxRequestTarget;
import wicket.contrib.dojo.markup.html.DojoLink;
import wicket.contrib.dojo.markup.html.Bubble.DojoBubble;
import wicket.markup.html.WebPage;

public class BubbleSample extends WebPage {

	public BubbleSample(PageParameters parameters){
		final DojoBubble bubble = new DojoBubble("bubble");
		add(bubble);

		add(new DojoLink("link"){
			public void onClick(AjaxRequestTarget target) {
				bubble.place(target, "stick");
				bubble.show(target);
			}
		});
	}
}



In this sample we attach the bubble on a specific markup id in the html with place(AjaxRequestTarget, String) method, but another place(AjaxRequestTraget, Component) exist to attach the DojoBubble on a Component.

Show and Hidde method allow to display/undisplay the bubble during a AjaxResponse. Bubble.place Method can be called several time to change the area where the bubble is displayed.
