Java API to work with the scriptaculous Effects library.

Documentation on the scriptaculous effects library can be found here:
http://wiki.script.aculo.us/scriptaculous/show/VisualEffects

This library is usually most useful when working with Ajax callbacks.

new AjaxLink(this, "clicker") {
  public void onClick(AjaxTarget target) {
    target.addJavascript(new Effect.Highlight(this).toJavascript());
  }
}
