Modal Dialog Feature

 Modal Dialog is a very simple widget allowing to show a div in popup and blocking the backgound. Modal comes with a closer and an oponer class to open or close it.
 

Java Sources

package wicket.contrib.dojo.examples;

import wicket.PageParameters;
import wicket.contrib.dojo.markup.html.dialog.DojoDialog;
import wicket.contrib.dojo.markup.html.dialog.DojoDialogCloser;
import wicket.contrib.dojo.markup.html.dialog.DojoDialogOpener;
import wicket.contrib.dojo.toggle.DojoWipeToggle;
import wicket.markup.html.WebPage;

public class DialogShower extends WebPage {

    public DialogShower(PageParameters parameters){
        DojoDialog dialog = new DojoDialog("dialogPanel");
        add(dialog);
        dialog.setToggle(new DojoWipeToggle(500));
        add(new DojoDialogOpener("openner", dialog));
        dialog.add(new DojoDialogCloser("closer", dialog));
    }
}

In this sample we create a DojoDialog containing a DojoDialogCloser. A DojoDialogOpener is also added to the page because the default behavior of a DojoDialog is to be hidden.

A DojoToggle is set on the DojoDialog. A DojoToggle is an effect added when a DojoWidget is shown/hidden. There are three avalaible DojoToggle :

  1. DojoExplodeToggle
  2. DojoFadeToggle
  3. DojoWipeToggle
