Overview

This project hosts examples of the Wicket-stuff projects. If you are curious what they do and how they work: check this project out.

This is also the much searched for "CD Application" found at: http://www.wicket-library.com/wicket-contrib-examples

Latest News

January 22, 2007 - Merging example projects

Work is underway to merge some of the examples projects in wicket-stuff with this project. The plan is to have just one examples project with examples of most of the wicket-stuff projects.
