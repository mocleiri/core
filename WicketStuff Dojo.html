Overview

The powerful and easy-to-use object-oriented web framework Wicket meets Dojo - a javascript toolkit which lets you make your webapps even more usable, responsive, and functional. This includes graphical decorations, AJAX functionality and other javascript enhancements.


Overview

This version is based on Dojo 0.4. There is an updated version based on Dojo 1.1: http://wicketstuff.org/confluence/display/STUFFWIKI/wicketstuff-dojo-1.1.

Latest News

August 07, 2007 - WicketStuff Dojo 1.3.0-beta is released 

The first new WicketStuffDojo release is out. It is named 1.3.0-beta in order to follow wicket version number. WicketStuffDojo-1.3.0-beta depends on wicket-1.3.0-beta2. You can find it on wicketstuff public repo 

Have good fun with it

The project has a new developer, welcome Grégory Maes!

April 18, 2007 - New WicketStuff Dojo developer Bart Molenkamp

The project has a new developer, welcome Bart Molenkamp!

April 11, 2007 - Project renamed to WicketStuff Dojo

As the SourceForge project is called WicketStuff, it makes sense to rename wicket-contrib-dojo to WicketStuff Dojo.  The package names have been updated to org.wicketstuff.dojo to reflect this change.

February 4, 2007 - Wicket-contrib-dojo is still alive

 Wicket-contrib-dojo is still alive... A lot of stuff have been added since January 2006. All new components and behaviors are wicket 1.X compatible, so we are waiting for Wicket 1.3 release to also release the new Wicket-contrib-dojo distribution. Even if you can not download a jar from sourceforge for the moment, sources are available on svn
