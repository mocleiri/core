TextField that provides static suggestions for autocompletion

Core scriptaculous documentation can be found here:
http://wiki.script.aculo.us/scriptaculous/show/Autocompleter.Local

Static Results

TextField color = new TextField("color");
name.add(new AutocompleteBehavior(new String[] {"red", "green", "blue"}));

Dynamic Results

uses Ajax

TextField color = new TextField("color");
name.add(new AjaxAutocompleteBehavior() {
  protected String[] getResults(String input) {
    //return custom results from user input
  }
});
