About

Wicket Openlayers Contribution enables you to create Openlayers Maps on your wicket page. Doing GIS the easy way.

Features

  • Add Markers
  • Add Markers with popups
  • Tested with FF2,Safari
  • Updates done via Ajax
  • Use your own map server!
  • Integration with JTS, Java Topology Service (http://www.vividsolutions.com/jts/jtshome.htm)
  • Draw Shapes (near future december - 2008)
