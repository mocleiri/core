Things we'd like:

  • Improved documentation for the modules that make up the Wicket Stuff project as a whole, including how to set them up, troubleshooting what they do, how they do it, how to use them and how to extend them.

Things that don't live here:

  • Articles and documentation about the Wicket Core or Wicket Extensions projects. These should go on Wicket's wiki instead.
  • Articles and information about integrating Wicket with other technologies or frameworks, unless it directly relates to code in the Wicket Stuff project. These should also go on Wicket's main wiki, as it's more likely people will find it there. You can always link to that wiki from here.
